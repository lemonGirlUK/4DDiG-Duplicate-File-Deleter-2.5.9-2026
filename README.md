## 🔧 Configuration

The tool saves settings to `%APPDATA%\4DDiG Duplicate File Deleter\config.ini`. You can manually customize scanning behavior by editing this file:

```ini
[Scan]
ExcludeFolders=C:\Windows;C:\Program Files;%TEMP%
MinFileSize=1MB
MaxFileSize=10GB
CompareMode=hash
AutoDelete=false
```
